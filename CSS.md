# CSS

<hr>

## CSS 란
- HTML로 작성된 웹페이지의 **레이아웃, 색상, 폰트, 크기 등 시각적인 스타일을 정의**하여 꾸며주는 스타일 시트 언어이다.
- 내용과 디자인을 분리하여 **웹 문서의 유지보수를 쉽게 하며, 반응형 웹 구현 등 화면 구성에 필수적**이다.

<hr>

## 주요기능
- __*색상 및 배경*__ : 글자 색(color), 배경 색상(background - color)을 이용하여 변경한다.
- __*레이아웃*__ : margin, padding, display, pasition을 활용하여 요소를 배치한다.
- __*폰트 및 텍스트*__ : 글꼴(font - family), 크기(font - size), 정렬(text -align)으로 설정한다.
- __*반응형 디자인*__ : 화면 크기에 따라 스타일을 다르게 적용한다.

<hr>

## 사용법
- __*인라인 스타일*__ : HTML 태그 내에 style 속성을 직접 작성한다.
- __*내부 스타일 시트*__ : HTML 문서 내 < style > 태그를 사용한다.
- __*외부 스타일 시트*__ : .css 파일을 별도로 만들어 HTML에 연결한다. (가장 권장된다)

<hr>

## 주요 특징
- 폭포처럼 **위에서 아래로 스타일이 적용**되며, 더 구체적인 스타일이 우선순위를 갖는 **계단식 구조**이다.
- 웹퍼블러셔, 프론트엔드 개발자, 디자이너에게 필수적인 기술로, HTML의 **제한적인 디자인 기능을 보완**하여
  화려하고 구조화된 웹사이트를 만든다.

<hr>

## CSS 기본 문법
- CSS의 문법은 크게 **선택자(selector)와 선언부(declaratives)로** 구성된다.
- 선택자는 CSS를 적용할 HTML 요소를 가리킨다.
- 선언부는 **하나 이상의 선언들을 세미콜론으로 구분**하여 포함할 수 있고, **중괄호를 사용하여 전체를 둘러싼다**.
- 각 선언은 **CSS 속성명과 속성값을 가지며, 그 둘은 콜론으로 연결**된다.
- 언제나 **마지막에 세미콜론**이 필요하다.
### 작성 예시 : a { background-color: yellow; font-xize: 16px; }

<hr>

## CSS 선택자의 종류
- HTML 요소 선택자
- 아이디 선택자
- 클래스 선택자
- 그룹 선택자

<hr>

## HTML 요소 선택자
- CSS 적용 대상으로 HTML요소의 이름을 직접 사용하여 선택할 수 있다.
- 작성 예시 :
             < style >

              h2 { color: teal; text-decoration: underline; }
             
             < /style >
             
## 아이디 선택자
- CSS를 적용할 대상으로 **특정 요소를 선택할 때** 사용된다.
- 웹 페이지에 포함된 여러 요소 중에서 **특정 아이디 이름을 가지는 요소만**을 선택한다.
- CSS에서 하나의 웹 페이지에 속하는 여러 요소에 같은 아이디 이름을 사용했을 때,  **HTML과 자바스크립트 작업시 문제가 발생**할 수 있다.
- 작성 예시 : < style >

      #heading { color: teal; text-decoration: line-through; }
  
             </style>
  
< h2 id="heading" > 이 부분에 스타일을 적용합니다.< /h2 >

## 클래스 선택자
- 특정 집단의 **여러 요소를 한번에 선택**할 때 사용한다.
- 이 특정 집단을 **클래스** 라고 하며, 같은 클래스 이름을 가지는 요소들을 모두 선택해준다.
- 작성 예시 : .headings { color: lime; text-decoration: overline; }

  < /style >


  < h2 class="headings" > 이 부분에 스타일을 적용합니다. < /h2 >

  < p > class 선택자를 이용하여 스타일을 적용할 HTML 요소들을 한 번에 선택할 수 있습니다. < /p >

  < h3 class="headings" >이 부분에도 같은 스타일을 적용합니다.</h3>

## 그룹 선택자
- 선택자 여러개를 하나로 묶는 선택이다.
- 결합자 :
  - 자손 결합자 : A 선택자 **하위에 있는 모든 B 선택자**를 선택한다.
  - 자식 결합자 : 첫번째 요소의 바로 아래 자식인 **태그를 선택**한다.
  - 일반 형제 결합자 : 선택자 A 뒤에 오는 **형제 관계 요소의 선택자 B의 모든 요소**를 선택한다.
  - 인접 형제 결합자 : 첫번째 요소의 **바로 뒤에 위치하며 같은 부모를 공유하는 두번째 요소**를 선택한다.

## CSS 박스 모델
- HTML 요소를 크게 **콘텐츠(Content), 패딩(Padding), 테두리(Border), 마진(Margin) 네 가지 영역으로 정의**하는 레이아웃 기본 개념이다.
- content : 태그 박스 안의 **내용, 텍스트나 이미지**
- padding : content 주위 영역 **(태그 안)**
- border : padding과 content를 둘러싼 주위 영역 **(태그와 밖의 경계)**
- Margin : **border의 밖** (태그와 태그 밖 바깥 태그나 body 사이의 영역)
