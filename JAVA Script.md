## JAVA Script란

- HTML/CSS와 함께 웹페이지를 구성하는 핵심 프로그래밍 언어이다.
- 사용자 인터렉션, 애니메이션, 데티ㅓ 처리 등 웹사이트에 동적인 기능(스크립트)을 부여한다.
- 클라이언트 (브라우저)측에서 작동하지만 Node.js를 통해 서버측에서도 사용할 수 있다.

<hr>

## 변수란

- 자바 스크립트에서 변수는 데이터를 담는 그릇을 의미한다.
- 할당 연산자(=)를 이용하여 값을 할당시킨다.
- 식별자인 변수명을 사용하여 변수에 저장된 값을 참조한다.
- 한 줄에 변수 선언과 값 할당을 할 수 있으며, 한번에 여러 변수를 선언할 수 있다.

<hr>

## 변수 선언 방식

## var

- ES5에서 변수를 선언할 수 있는 유일한 방법으로 한 번 선언된 변수명에 데이터를 재할당, 동일한 변수명으로 재선언이 가능하다.
- 함수 레벨 스코프 : 함수 내부에서 선언된 함수만 지역변수로 한정하고, 함수 밖에서 선언한 var 변수는 전역 변수로 간주하여 외부에서도 접근이 가능하다. (이로 인한 문제가 발생할 수 있다.)
- 스코프 : 유효한 참조범위의 함수 내에서 선언된 변수는 함수 내에서만 유효하며 함수 외부에서는 참조할 수 없다.
- 블록레벨 스코프 : 모든 코드 블록내에서 선언된 변수는 코드 블록 내에서만 유효하며 코드 블록 외부에서는 참조할 수 없다.
- var 키워드 생략 가능 : var의 생략을 허용함으로써 의도하지 않은 변수를 전역화 시킨다.

<hr>

## let

- 한번 선언된 변수명에 데이터를 재할당은 가능하나 동일한 변수명으로 재선언할 수 없다.
- 동일한 변수명으로 재선언이 불가해 변수명이 동일하여 발생할 수 있는 문제를 방지할 수 있다.
- 변수 호이스팅이 발생하지 않음 : let 키워드로 선언된 변수를 선언문 이전에 참조하면 참조 에러가 발생한다.
- 블록레벨 스코프 : 모든 코드 블록내에서 선언된 변수는 코드 블록 내에서만 유효하며 코드 블록 외부에서는 참조할 수 없다.

<hr>

## const

- constant : 변하지 않는 값 선언 시 사용한다.
- 변수를 선언하는 시점에 값을 할당하고, 값을 재할당하거나 동일한 변수명으로 재선언 둘다 불가능하다.
- let과 마찬가지로 호이스팅이 발생하지 않으며, 블록 레벨 스코프를 지원한다.

## 조건문이란

- 특정 조건을 만족할 때 실행하는 명령의 집합 이며, 어떤 작업을 수행하고 싶을 때 사용하는 것이다. -특정 조건 충족시에만 특정 코드 블록을 실행하기 때문에 흐름을 효과적으로 제어할 수 있고, 프로그래밍적 유연성이 향상된다.

## 조건문의 종류

## if문

- 조건식을 만족일때만 코드를 실행하고 만족하지 않다면 else문이 실행된다.
- else문은 선택사항이며 작성하지 않을 경우 if문만 체크 후 다음 코드로 넘어간다.

## else if문

- 여러개의 조건을 검사할 때 사용할 수 있다.
- 횟수를 무한정 늘릴 수 있으며 하나의 조건을 만족하면 코드를 실행하고 나머지 else if, else 조건문을 무시한다.
- else문은 선택사항으로 작성하지 않아도 된다.

## 삼항연산자

- 참/거짓 하나의 조건문만 사용할 때 삼항연산자를 사용하면 한 줄로 표현할 수 있다. -조건 : 평가되는 조건이다. 참이면 값1, 거짓이면 값2를 반환한다.
- 값1 : 조건이 참일 경우 반환된다.
- 값2 : 조건이 거짓일 경우 반환된다.

## 함수란

- 다른 객체처럼 속성 및 메소드를 가질 수 있어서 일급 객체이며 인자를 가질 수 있는 코드블록이다.
- 자체 범위를 가지고, 부모함수의 지역변수에 접근할 수 있다.

## 함수를 사용하는 이유

- 코드를 재사용할 수 있다.
- 다른 인자를 사용하여 동일한 코드를 여러번 재사용하고 다른 결과를 도출할 수 있다.

## 함수의 특징

- 변수안에 담길 수 있다.
- 객체의 속성안에 메소드로 담길 수 있다.
- 다른 함수의 인자값을 전달될 수 있다.
- 함수의 return값으로도 사용할 수 있다.
- 배열의 값으로도 사용할 수 있다.

## 함수 구문

- 함수 내에 선언된 변수는 함수의 지역변수가 된다. 함수 내에서만 접근할 수 있고 함수가 시작될 때 생성되며 함수가 완료되면 삭제된다.
- 함수 이름에는 문자, 숫자, 밑줄 및 달러 기호가 포함될 수 있다.
- 괄호에는 쉼표로 구분된 매개 변수 이름이 포함될 수 있다.
- 함수가 실행할 코드는 { }안에 있다.
